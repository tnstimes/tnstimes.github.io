About TNS Times

TNS Times is a new formal set of issues from Super Interviews. Issues will only be availible online and will be released on this website when availible.
Every issue contains news, puzzles, stories and more! If you are lucky, you might even get to check out some cooking recipes!

If you would like to join TNS Times as an editor or writer, please approach Xinyuan (5-3), Jun Ying (5-3) or Andre (5-4).
